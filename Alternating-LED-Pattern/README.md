# Alternating LED Pattern

## Overview

This project demonstrates an alternating LED lighting pattern using Arduino and 8 LEDs.

## Components Used

* Arduino Uno
* 8 LEDs
* Resistors
* Breadboard
* Jumper Wires

## Working

The LEDs are divided into two groups. One group turns ON while the other remains OFF. After a delay, the states are reversed, creating an alternating lighting pattern.

## Applications

* Decorative Lighting
* Signal Indicators
* Learning Digital Output Control
* LED Pattern Generation

## Skills Learned

* Arduino Programming
* Digital Output Control
* Timing Functions
* Sequential Logic
* Circuit Design
