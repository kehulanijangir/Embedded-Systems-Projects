# Buzzer and LED

## Overview

This project demonstrates the use of a buzzer and LED as an alert system using Arduino.

## Components Used

* Arduino Uno
* Buzzer
* LED
* Resistor
* Breadboard
* Jumper Wires

## Working

The Arduino controls both the buzzer and LED. When activated, the LED provides a visual indication while the buzzer generates an audible alert signal.

## Applications

* Alarm Systems
* Security Systems
* Warning Indicators
* Notification Systems

## Skills Learned

* Buzzer Interfacing
* LED Control
* Digital Output Programming
* Circuit Design
* Arduino Programming
