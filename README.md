# Embedded-Systems-Projects
My embedded systems projects and experiments.

This repository contains embedded systems projects developed using Arduino and Tinkercad as part of my engineering learning journey. These projects helped me understand hardware-software interaction, sensor interfacing, circuit design, and embedded programming fundamentals.

## Projects Included

### LED Blinking
A basic Arduino project demonstrating LED control and timing functions.

### Temperature Simulator
Simulation of temperature monitoring using Arduino and temperature sensors.

### LED and Potentiometer
Control of LED brightness using a potentiometer and PWM.

### Soil Moisture Monitor
Monitoring soil moisture levels using sensors and Arduino.

### Seven Segment Display
Displaying numerical digits using a seven-segment display.

### NeoPixel Strip Lights
Generation of colorful lighting effects using NeoPixel LEDs.

### Alternating LED Pattern
Implementation of alternating LED sequences using 8 LEDs.

### Buzzer and LED
A simple alert system using visual and audio indicators.
