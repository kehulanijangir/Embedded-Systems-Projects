# LED Blinking

## Overview
This project demonstrates basic LED blinking using Arduino and Tinkercad.

## Components Used
- Arduino Uno
- LED
- Resistor
- Breadboard
- Jumper Wires

## Working
The LED turns ON for 1 second and OFF for 1 second continuously using Arduino code.

## Applications
- Learning Arduino Basics
- Understanding Digital Output
- Embedded Systems Fundamentals

## Skills Learned
- Arduino Programming
- Digital Output Control
- Circuit Design
- Timing Functions
