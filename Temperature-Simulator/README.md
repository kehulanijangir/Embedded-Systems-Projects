# Temperature Simulator

## Overview

This project simulates temperature monitoring using Arduino and a temperature sensor in Tinkercad.

## Components Used

* Arduino Uno
* Temperature Sensor
* Breadboard
* Jumper Wires

## Working

The temperature sensor measures temperature values, and the Arduino reads and processes the data. The temperature can then be monitored through the simulation.

## Applications

* Environmental Monitoring
* Weather Stations
* Smart Home Systems

## Skills Learned

* Sensor Interfacing
* Analog Input Reading
* Arduino Programming
* Data Processing
