# Seven Segment Display

## Overview

This project demonstrates displaying numbers on a seven-segment display using Arduino.

## Components Used

* Arduino Uno
* Seven Segment Display
* Resistors
* Breadboard
* Jumper Wires

## Working

The Arduino controls the individual segments of the display to show numerical digits. Different combinations of segments are activated to represent numbers from 0 to 9.

## Applications

* Digital Clocks
* Counters
* Scoreboards
* Electronic Measuring Instruments

## Skills Learned

* Display Interfacing
* Digital Output Control
* Circuit Design
* Arduino Programming
* Embedded Systems Fundamentals
