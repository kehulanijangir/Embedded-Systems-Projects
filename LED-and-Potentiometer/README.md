# LED and Potentiometer

## Overview

This project controls the brightness of an LED using a potentiometer.

## Components Used

* Arduino Uno
* LED
* Potentiometer
* Resistor
* Breadboard

## Working

The potentiometer acts as a variable resistor. The Arduino reads its value and adjusts the LED brightness accordingly using PWM.

## Applications

* Light Dimming Systems
* Brightness Control
* User Input Devices

## Skills Learned

* PWM Control
* Analog Input Reading
* LED Interfacing
* Arduino Programming
